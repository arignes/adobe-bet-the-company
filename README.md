# Adobe Bet the Company — Group 2

An interactive scroll-driven case study: **Adobe's 2013 transition from perpetual licenses (Creative Suite) to subscriptions (Creative Cloud)**, FY2008–FY2024.

Built for *Measuring Value for Strategic Decision Making* — Group 2:
Mohammed Ramzi Altaylouni · Sofia Cunha · Daniela Espino Mendoza · Nathalie Henkens · Arina Nesterenko · Ahmed Sanad

## What's inside

- 15 full-screen sections with a year-rail navigation and scroll-reveal animations
- Interactive SVG charts (revenue & margin J-curve, ARR growth, ROIC vs WACC spread, share price) with hover tooltips
- Interactive financial tables with column pinning
- Pure HTML/CSS/JS in a single `index.html` — no build step, no dependencies

## Run it

Open `index.html` in any browser, or serve it:

```bash
python3 -m http.server 8000
```

## Sources

Adobe 10-K filings FY2008–FY2024, investor presentations, earnings calls. Figures marked ≈ are estimates pending final balance-sheet inputs.
